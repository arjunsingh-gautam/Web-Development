In this Lesson we will learn about List elements in HTML

Types of List

  1. Unordered List
  2. Ordered List
  3. Definition List
  4. Description List
  5. Nested List

Use of Each List Type:

Unordered List
Used when order is not important
Ordered List
Used when order is important
Definition List
Used when there term definition relationship is present
Description List
Used when we don't want to use stict term definition relationship
Nested list
Used when we want to represent a hierarchial structure

When to use list elements

  • To present data in a order or bullet points
  • To create navigation bar
  • For structuring content
